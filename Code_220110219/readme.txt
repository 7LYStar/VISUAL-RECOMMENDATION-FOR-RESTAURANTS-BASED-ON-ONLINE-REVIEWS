The data for this project is too large to submit via ness. 
You can download the data at the following URL.
https://www.yelp.com/dataset

This project contains a total of 6 code files and a visualization file.
The code needs to be run in the Google colab environment.
The code files can be executed in the following sequence:

1.json_to_csv_business.ipynb     This code converts json files to csv files for later use.
2. data_process.ipynb
3.LDA.ipynb
4.LSTM.ipynb
5.sentiment score.ipynb
6.business_topic.ipynb

The visualization file requires installing the tableau visualization software to work correctly.
